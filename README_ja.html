React 19 FastAPI PostgreSQL Redis Docker GitHub Actions

🎯 背景はいけい・目的もくてき

大学生だいがくせいが学習がくしゅう・就職活動しゅうしょくかつどう・校内生活こうないせいかつで蓄積ちくせきした経験けいけんを 共有きょうゆうできる専用せんようプラットフォームが存在そんざいしませんでした。 MemoLucky は「経験談けいけんだんの投稿とうこう」「学内がくないフリマ」「通知つうち」を 1つの Web アプリに統合とうごうし、複数ふくすうのサービスを行き来いききする手間てまを解消かいしょうします。

✨ 主な機能おもなきのう

機能きのう説明せつめい
📝 経験談けいけんだん投稿とうこうタグtag・匿名とくめい・画像添付がぞうてんぷ・多言語たげんご対応たいおう
🛍️ フリマ商品出品しょうひんしゅっぴん・購入申請こうにゅうしんせい・出品者しゅっぴんしゃとのチャットchat
🔔 通知つうちいいねいいね・コメントcomment・フォローfollow・メッセージの即時通知そくじつうち
🏆 バッジ初投稿はつとうこう・連続投稿れんぞくとうこうなど 8種類しゅるいの実績じっせき
💬 DM商品売主しょうひんうりぬしとの直接ちょくせつメッセージ
🌍 多言語たげんご日本語にほんご / English 切替きりかえ

🚀 起動方法きどうほうほう

git clone https://github.com/syokan00/Database_Final.git
cp backend/env.example backend/.env
docker compose up -d
cd frontend && npm install && npm run dev

起動後きどうご、http://localhost:5173 でアクセス可能かのうです。
