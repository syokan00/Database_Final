# ペルソナ（Persona）

## ペルソナ 1：山田 太郎（やまだ たろう）

- 年齢：21 歳  
- 職業：大学 3 年生（情報系学部）  
- 居住地：首都圏で一人暮らし  
- 利用デバイス：ノート PC、スマートフォン  

### 背景・状況

- 授業・アルバイト・サークル活動などで、毎日スケジュールが詰まっている  
- 課題の締切、買い物リスト、ちょっとしたアイデアなどをメモしたいが、  
  スマホのメモアプリや紙のノートなど複数に分散してしまっている  
- 前期に使った教科書や参考書など、「捨てるのはもったいないが使わないもの」が溜まりつつある  

### 目標

- 日々のタスクやアイデアを、**1 つの場所に整理しておきたい**  
- 使わなくなった参考書や小物を、**同じ学生同士で気軽に譲りたい / 売りたい**  
- 難しい設定なしで、ブラウザからすぐにアクセスできる Web アプリがほしい  

### 課題・ペイン（Pain）

- メモが「紙のノート」「スマホメモ」「PC のテキストファイル」などに分散し、探すのに時間がかかる  
- フリマアプリは便利だが、大学内のちょっとしたやり取りにはオーバースペックに感じる  
- 友人との間で「この本ほしい人いない？」と毎回 SNS でやりとりするのが面倒  

### 本システムから得られる価値

- ブラウザだけでアクセスでき、**メモと出品を同じ UI 上で管理できる**  
- 「勉強のメモ」と「物のやり取り」が一つのアプリで完結するため、管理が楽になる  
- 将来的に大学・サークル内だけで共有する仕組みを作ることで、身近なコミュニティでのやり取りに最適なツールとなる

## ペルソナ 2：佐藤 花子（さとう はなこ）

- 年齢：20 歳  
- 職業：短期大学 2 年生（デザイン系）  
- 居住地：実家暮らし  
- 利用デバイス：タブレット、スマートフォン  

### 背景・状況

- レポート課題や作品制作のアイデアを、よく手書きメモや画像キャプチャに保存している  
- 授業ごとに必要な道具が多く、「どの授業で何が必要だったか」を管理しきれない  
- 下級生に道具を譲りたいが、うまく告知する場がない  

### 目標

- 授業ごとの重要ポイントややることリストを、科目ごとに整理して残したい  
- 使わなくなった画材や道具を、簡単に「欲しい人」に届けたい  

### 課題・ペイン

- SNS で毎回「誰か要りませんか？」と投稿するのは気が引ける  
- 道具をあげたい・売りたいタイミングと、欲しい人のタイミングがなかなか合わない  

### 本システムから得られる価値

- 授業ごとのメモを残しつつ、関連する道具の出品も同じ場所で管理できる  
- 将来的に「授業名」や「カテゴリ」で検索できるようになれば、  
  下級生とのマッチングもしやすくなる
